Lucid 개인정보처리방침

시행일자: 2025년 2월 10일 | 최종 수정일: 2025년 2월 10일

Lucid(이하 "앱")은 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 개인정보처리방침은 앱이 수집하는 개인정보의 항목, 수집 목적, 보유 기간 및 이용자의 권리에 대해 안내합니다.

1. 수집하는 개인정보 항목 및 수집 방법

가. 회원가입 및 로그인 시

수집 항목 수집 방법 필수 여부
이메일 주소, 비밀번호 이메일 회원가입 시 이용자 직접 입력 필수
이메일 주소, Google 계정 식별 정보 Google 로그인 시 자동 수집 필수
Firebase 사용자 고유 ID (UID) 계정 생성 시 자동 생성 필수

나. 서비스 이용 과정에서

수집 항목 수집 방법 필수 여부
꿈 내용 (텍스트) 이용자 직접 입력 또는 음성 인식을 통한 변환 필수
꿈 키워드 (감정, 장소, 행동 등) 이용자 선택 입력 선택
꿈 기록 날짜 자동 기록 필수
AI 해몽 분석 결과, 일일 조언 AI 분석 시 자동 생성 필수
AI 생성 꿈 이미지 AI 이미지 생성 시 자동 생성 (기기 내 저장) 선택
앱 내 재화(달조각) 정보, 출석 기록 서비스 이용 시 자동 기록 필수

다. 음성 녹음

앱은 꿈 내용을 음성으로 기록하기 위해 마이크 접근 권한을 요청합니다. 음성 데이터는 기기 내에서 텍스트로 변환되며, 음성 원본은 서버에 전송하거나 저장하지 않습니다.

2. 개인정보의 수집 및 이용 목적

  • 회원 관리: 회원 식별, 로그인 인증, 계정 관리
  • 서비스 제공: 꿈 일기 기록 및 저장, AI 기반 꿈 해몽 분석, 꿈 시각화 이미지 생성
  • 통계 및 분석: 이용자 개인의 꿈 패턴 통계 제공 (꿈 빈도, 키워드 분석 등)
  • 재화 시스템 운영: 달조각(앱 내 재화) 적립 및 사용 관리, 출석 보상 제공
  • 광고 제공: Google AdMob을 통한 보상형 광고 노출

3. 개인정보의 보유 및 이용 기간

이용자의 개인정보는 회원 탈퇴 시 또는 수집 목적 달성 시 지체 없이 파기합니다.

  • 회원 정보: 회원 탈퇴 시까지
  • 꿈 기록 데이터: 회원 탈퇴 시까지 (이용자가 개별 삭제 가능)
  • 기기 내 저장 데이터: 앱 삭제 시 자동 삭제

단, 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 보관합니다.

4. 개인정보의 제3자 제공

앱은 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 의해 요구되는 경우

5. 개인정보 처리 위탁

앱은 서비스 제공을 위해 다음과 같이 개인정보 처리를 위탁하고 있습니다.

위탁받는 자 위탁 업무 내용
Google LLC (Firebase) 사용자 인증, 클라우드 데이터 저장 (Firestore)
Google LLC (Gemini API) 꿈 내용 분석 및 해몽 결과 생성
Google LLC (Imagen API) 꿈 시각화 이미지 생성
Google LLC (AdMob) 보상형 광고 제공

위탁받는 자의 개인정보 처리에 대한 자세한 사항은 Google 개인정보처리방침을 참고하시기 바랍니다.

6. 이용자의 권리 및 행사 방법

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 개인정보 열람 요청: 앱 내 저장된 꿈 기록 및 계정 정보 확인
  • 개인정보 수정 요청: 계정 정보 변경
  • 개인정보 삭제 요청: 개별 꿈 기록 삭제 또는 회원 탈퇴를 통한 전체 삭제
  • 처리 정지 요청: 개인정보 처리 정지 요청

권리 행사는 앱 내 기능을 통해 직접 수행하거나, 아래 연락처로 요청하실 수 있습니다.

7. 개인정보의 안전성 확보 조치

앱은 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다.

  • 데이터 암호화: Firebase Authentication을 통한 비밀번호 암호화 저장
  • 안전한 통신: HTTPS 프로토콜을 통한 암호화 통신
  • 접근 권한 관리: Firebase 보안 규칙을 통한 데이터 접근 제한 (본인 데이터만 접근 가능)
  • 기기 내 저장: 음성 데이터는 기기 내에서만 처리되며 외부 서버로 전송하지 않음

8. 자동 수집 장치의 설치·운영 및 거부

앱은 웹 쿠키를 사용하지 않습니다. 다만, Google AdMob 광고 SDK가 광고 개인화를 위해 광고 식별자(GAID)를 수집할 수 있으며, 이용자는 기기 설정에서 광고 추적을 제한할 수 있습니다.
